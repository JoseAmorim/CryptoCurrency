## Installation

git clone git@github.com:JoseAmorim/CryptoCurrency.git

cd CryptoCurrency

yarn install

Go to https://coinmarketcap.com/api/ then pick the api key

## Usage

cd CryptoCurrency

react-native run-android or react-native run-ios(just if you have macOS)
